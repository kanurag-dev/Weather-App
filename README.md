# Weather App

A simple weather application built using HTML, CSS, and JavaScript. It fetches real-time weather information using the OpenWeather API.

## Features

* Search weather by city name
* Display current temperature
* Display weather condition
* Display humidity
* Display wind speed
* Display weather icon
* Press **Enter** to search
* Handles empty input and invalid city names

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeather Geocoding API
* OpenWeather Current Weather API

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter a city name and click **Search**.
